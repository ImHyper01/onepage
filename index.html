Agent Rex - De uitleg

Concept

Agent Rex is de karakter in de game die oneindig blijft rennen. Onder tussen wanneer hij aan het rennen is moet ie reken sommen oplossen, om zo door te kunnen gaan met de game. De reken sommen zullen zelf door de speler gemaakt worden want je gaat geldzakjes zien die bijvoorbeeld een + of – teken erop heeft zitten, als je een van die geldzakjes pakt, dan veranderd de som. Dus je hebt in eigenhanden hoe moeilijk de som gaat worden. Hoe moeilijker de som hoe meer punten je gaat krijgen. Agent rex zou ook obstakels moeten ontwijken om zo door te kunnen gaan. Als die 3x wordt geraakt eindigt de game en ben je af

Onderbouwing

Ons concept idee is een Subway surfer type game, met een reken twist er in. Onze doelgroep gaan scholieren zijn van groep 5-6. We willen ze een leuke game geven, waardoor rekenen leuker wordt voor hun, want eerlijk gezegd wilt niemand een saai reken les volgen. Daarom hopen we dat deze game dat gaat veranderen voor hun.

Controls

De controls van de game gaan zijn de pijltjes toets naar ← of → Het zijn makkelijke controls, dus iedereen kan het spelen

Waar speelt het zich af?

De game gaat zich afspelen in de jungle, daarom ook deze achtergrond, de obstakels zullen rotsen worden en de + en – zakjes zullen geld zakken worden
